Setiware
Introduction What means Setiware ?
When you work for a long time and produce something people like, it's a nice idea to give the package the shareware option. But instead of claiming money, I would rather love to get more Seti@Home units as I'm an E.T tracker ;) So Setiware is a shareware variante.
SETI@home is a scientific experiment that uses Internet-connected computers in the Search for Extraterrestrial Intelligence (SETI). You can participate by running a free program that downloads and analyzes radio telescope blocks of data. The results of analyzed block of data are sent back when processed and give you one credit.
The more credit you have, the more search you have done. As a Seti addicted, I am looking for more processing power to analyze more data. So, few Seti@home units against the use of w3perl seems to be fair for me. You don't have to but it would be nice of you. Just logon to their server and reduce the data using my logging account : domisse@w3perl.com... :)

Condition of use The w3perl package is SETIWARE.

It means you can use it as long as you want. If you like the package, you can (but you don't have to) help me in a various ways :
  • you can help me to promote or develop the package
  • you send me a small fee
  • you can send me a postcard (except from Paris please ;)
  • you can compute some SETI@Home credits for me.

If you choose one of this option, I'll be glad to hear from you as a registered user. This allow you to use the latest bug fix, to receive email support, to suggest me features you would like to see.
If you now are using an improved version of w3perl, is especially because there some people who encouraged me to continue its development by sending me a registration fee or greetings.
