W3Perl scripts
All the files included in the w3perl package are listed below along with their primary use. Most of the files are needed for W3Perl to operate (optionnal are (*)).
Scripts with the sh extension are only available for Unix.

Files distributed :

Install and admin scripts
install.pl Fix Perl and directory paths (installation)
config.pl Configuration file (generated by the admin part)
Each script load a configuration file where your own parameters are stored
cron-w3perl.pl edit and install in the crontab to launch the scripts (installation) (*)
All scripts will be launch according to your time schedule.
squeezelog.sh Compress your log files each month (*)
cron-stats.sh Download your stats files and compute stats them locally (*)
confstats.pl Generate configuration files (admin) (*)
Update and build configuration file via a form.

Stats scripts
cron-pages.pl Main prog to initialise stats
cron-inc.pl Incremental stats
cron-hour.pl Hourly stats
cron-day.pl Daily stats
cron-week.pl Weekly stats
cron-month.pl Monthly stats
cron-url.pl URL mapping and Web's HTML files stats (*)
cron-session.pl Session stats (*)
cron-refer.pl Referer stats (*)
cron-agent.pl Agent log stats (*)
cron-error.pl Error log stats (*)
cron-history.pl Show CPU usage (*)

Documentations and Resources files
HTML files HTML documentation (english and french)
config files Configuration file examples
pays-conv Translation for countries codes
resources Icons and translation files
