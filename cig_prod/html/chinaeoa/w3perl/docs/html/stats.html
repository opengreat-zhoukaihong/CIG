W3Perl stats
I find quite funny to make some stats about w3perl. The package is now five year's old....enough to compute some data.

W3Perl size

The first decrease at the beginning is due to the use of the Fly graphic package instead of the more complex Gnuplot, giftrans and ppmtogif packages.
The second is about the implementation of the multi-language and multi-platform support wich reduce the size of the package.

With the 2.50 release, the package has reached the 2 Mo of data ! I was far to imagine five years ago that it could reach this size.
