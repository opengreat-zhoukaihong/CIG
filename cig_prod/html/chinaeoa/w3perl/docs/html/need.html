What you need
W3Perl You should first download the last package available. W3Perl is setiware.

Perl All programs have been writing in Perl... so you need a perl interpreter ! The package have been tested from Perl 4.0 to the latest Perl 5.6.0 You could have some problems with earlier version. Most Unix stations have already this language available. NT server should use ActivePerl 5.6

Fly 1.6.5 This package is the perl translation of the famous gd package. It enable you to produce gif on the fly from a perl script. All w3perl graphics are generated by Fly.
Easy to install, the program is written in C with a makefile included.
Binary and source are available also on this mirror site (USA).
Fly is an australian package made by Martin Gleeson.
