W3Perl Vous pouvez télécharger la dernière version. W3Perl est une application setiware.

Perl Toutes les scripts sont écrits en Perl... , vous devez donc posséder un interpréteur Perl !
L'application a été testée depuis la version Perl 4.0 jusqu'à la version courante 5.004. Vous pouvez néanmoins avoir quelques problèmes avec de vieilles versions. Sur NT, vous devez installer ActivePerl 5.22.

Fly 1.6.5 Cette application est la traduction en Perl du fameux logiciel gd. Elle permet de produire des images au format gif directement à partir d'un script Perl.
Facile à installer, le programme est écrit en C avec un Makefile inclus.
Les binaires ainsi que les sources sont disponibles sur ce site mirroir (USA).
Fly est un logiciel australian écrit par Martin Gleeson.
