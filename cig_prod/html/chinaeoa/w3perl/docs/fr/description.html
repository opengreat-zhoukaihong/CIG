Description de l'application
Le but de cette application est de produire des statistiques d'utilisation de votre serveur Web d'une maière simple et puissante. Pour cela, il est configurable à souhait.

Les informations sont diposées de maniès textuelles et graphiques pour permettent une meilleures analyses des résultats. J'espère que cet outil vous permettra de rendre votre site Web encore plus performant et efficace.

Caractéristiques principales
  • Fichiers de sortie en HTML
  • Rapide (fonctionne de manière incrémentale)
  • Réellement incrémentale (les relations entre variables sont sauvegardées)
  • Multi-langage (vous pouvez ajouter le votre facilement)
  • Différents types de sorties graphiques et textuelles
  • Statistiques heures par heures, journalières, hebdomadaires et mensuelles
  • Statistiques sur les pays, sites, pages, répertoires, domaines...
  • Résultats concis ou très détaillées
  • Fichier de configuration par serveur (utilisez autant de fichiers de configuration que vous le voulez)
  • Nombreuses options disponibles par lignes de commandes
  • Fonctionne sur tous les serveurs que j'ai pu rencontrés (Apache, NCSA, CERN, IIS, Netscape...)
  • Fonctionne avec des fichiers de log au format common, extended common, new extended ou IIS
  • Pas d'accès root ou d'accès cgi-bin nécessaire. Peut être utilisé depuis votre propre répertoire.
  • Fonctionne avec des fichiers de log compressés ou/et avec des fichiers de log mensuels/quotidiens
  • Peut être lançé chaque nuit de manière automatique par une crontab
  • Traduction des URL grace aux titres des documents HTML
  • Statistiques sur les références et les fureteurs
  • Statistiques sur les mots clés utilisés par les moteurs de recherche
  • Statistiques sur les erreurs
  • Statistiques sur les scripts
  • Statistiques sur les sessions
  • Statistiques sur les accès protégés par mot de passe
  • Statistiques sur votre site Web (arborescence, mauvais liens...)
  • Fonctionne sur tous types d'ordinateurs (Unix, NT, Mac et Acorn testés)
  • Focntionne avec la plupart des format de fichiers de logs (vous pouvez en ajouter d'autres facilement)
  • Support pour les serveurs virtuels dans les fichiers de log au format NECLF ou au format CLF et ECLF modifiés
  • Interface d'adminstration à distance
  • Supporte les noms de fichiers de log quotidiens ou mensuels
  • Pas d'accès telnet nécessaire pour utiliser l'application
  • Statistiques journalières très précises.

    • Structure

    Les dernières versions de l'application stockent les informations dans des répertoires de la manière suivante :

    data   Structure temporelle
    graph   Tous les graphes
    inc   Données incrémentales
    tmp   Utilisés pour la génération des graphes
    uk date Pages HTML sur les dates
      days Pages HTML sur les jours
      document Pages HTML sur votre site Web
      frame Pages HTML pour les frames
      list Pages HTML sur les erreurs, fureteurs, références et les répertoires
      page Pages HTML sur certaines pages
      pays Pages HTML sur les pays et les domaines
      session Pages HTML sur les sessions
      site Pages HTML sur les sites
      script Pages HTML sur les scripts
