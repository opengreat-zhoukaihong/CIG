Première utilisation



Si vous pouvez utiliser cron-w3perl.pl, allez y ! Lancer cron-w3perl.pl -h pour voir les différentes options disponibles.

Si vous ne pouvez pas utiliser cron-w3perl.pl (sites sous NT), suivez les instructions suivantes pour exécuter les scripts dans le bon ordre. Ne lancer pas tous les scripts en même temps car certains utilisent des fichiers temporaires ou des sorties d'autres scripts.

utilisez l'intrface d'administration à distance pour lancer les scripts excepté bien sur si vous n'avez pas d'accès à un répertoire cgi-bin.

Test de sortie
 		Lancezuniquement cron-hour.pl afin de vérifier qu'aucune erreur n'est survenue dans votre configuration.
Ensuite lancez cron-pages.pl en utilisant un petit fichier de log afin de ne pas attendre un long moment avant de vous rendre compte d'un problème éventuel. (ou vous pouvez spécifier un intervalle de quelques jours avec cron-pages en utilisant les options disponibles : voir cron-pages.pl -h pour l'aide en ligne)

Modifiez votre fichier de configuration pour changer quelques valeurs, ajouter des répertoires, peaufiner votre tri...et relancer cron-pages.pl
Lorsque votre paramérisation est parfaite, vous êtes prêt pour lancer l'ensemble des scripts dans le tableau 'First use' de l'interface d'administration. Si par la suite, vous modifier le fichier de configuration, vous devrez à nouveau relancer cron-pages (sauf par exemple, si seules les valeurs liées à l'aspect graphique sont altérées).

Uitilisation
quotidienne
 		Il n'est pas nécessaire de lancer tous les scripts tous les jours. Les stats sur les mois ne sont utilement calculées que le premier jour de chaque mois, les stats sur les semaines qu'une fois par semaine. Les stats sur votre site Web (cron-url.pl) peuvent n'être lancées qu'une fois par semaine si vos mises à jour de votre site Web n'est pas journalière. Referer, agent et error peuvent être exécutés tous les jours car ils tournent de manière incrémentale et que leurs informations sont vraiment pertinentes.

cron-inc.pl Calcule les stats incrémentales quotidiennes
