Version de démonstration
La version de démonstration est désormais la 2.40 (27 Juin 1998) De nouvelles fonctionnalités sont disponibles depuis la 2.00. Cette version est gratuite et fonctionne sous NT.
C'est une version non bridée sans temps limite d'utilisation, ni de restriction en temps de calcul..mais l'installation est plus simple avec la version enregistrée (surtout sous NT).
La prochaine version de démonstration sera disponible ... avant la fin de l'année.
N'oubliez pas que la licence n'est pas chère (300 FF maximum ;)

Nouveautés depuis la 2.30
  • Ne nécessite pas un accès telnet sur votre machine
  • Interface d'administration à distance disponible
  • Lancement automatique de tous les programmes (Unix)
  • Support pour les fichiers de log compressés
  • Installation sous NT simplifiée
  • Fichiers de log journaliers suuportés
  • Plus auncune restriction sur le format des fichiers de log
  • Stats sur les scripts calculées
  • Stats sur les références et les fureteurs calculées
  • Stats sur les moteurs de recherche calculées
  • Support pour plusieurs fichiers de configuration
  • Multi-langage disponible.
  • Stats sur plusieurs années possibles.

    • Limitations
  • Stats sur les semaines et le mois non calculées
  • Stats sur votre serveur non calculées :
    • Nombre de liens, d'images par pages, de répertoires, poids des pages.
    • Traduction d'URL, pages sans liens, pages locales, WIDTH, HEIGHT et ALT tags.
    • Liens érronnés, liens symboliques et détection des liens en absolus.
    • Pages les plus récentes, arborescence du site
  • Langue anglaise fournie uniquement
  • Stats sur les sessions non calculées :
    • Temps de connection
    • Nombre de viiteurs
    • Sessions multiples
    • Description des sessions
    • Balayage des moteurs de recherche
    • Charge moyenne par heures et par jours.
    • Stats sur les connexions par mots de passe.
  • Stats sur les erreurs non calculées
  • Pas de support disponible.
