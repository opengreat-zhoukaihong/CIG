Installation
Vous devez en premier lieu installer le logiciel Fly qui vous permettra d'obtenir des graphes pour w3perl. Ensuite, l'installation ne consiste qu'en l'edition de 3 lignes dans un fichier de configuration et l'utilisation d'une interface d'adminstration pour paramétrer vos sorties.

Fly
 		1 - Téléchargez le logiciel Fly sur Australie ou depuis un site mirroir aux USA (source et binaire disponibles)

2 - Décompressez le fichier téléchargé (tar -xzvf fly.tgz sous Unix ou unzip fly.zip sous NT)

3 - Compilez le logiciel ou utilisez l'exécutable téléchargé.

W3Perl

La procédure d'installation peut être diviséee en 5 parties différentes :

    1 - Installation des fichiers

L'application peut aussi bien être installée aussi bien dans le répertoire associé à votre serveur ou dans votre répertoire cgi-bin. L'installation standard suppose que vous pouvez créer un sous répertoire /w3perl/ au sein de votre répertoire cgi-bin. Vous devez éditer les trois premières lignes d'un script et le lancer pour terminer l'installation.

J'ai trouvé trois manières différentes d'installer l'application :

Telnet Cgi-bin Crontab
Hébergeur Non Oui Optionnel
Utilisateur Oui Non Optionel
Webmaster Oui Oui Oui

    2 - Edition du fichier de configuration

Les fichiers de configuration peut être construit avec l'aide d'un script si vous avez accès à un répertoire cgi-bin ou alors à la main.

    3 - Initialisation

Lancement de quelques scripts afin de vérifier votre fichier de configuration et de valider vos choix de sortie. Toutes les stats peuvent être lancées de manière automatique avec un seul script si vous êtes sous un serveur Unix.

    4 - Utilisation de la crontab

Ajoutez un script dans votre crontab pour que les statistiques soient lancées de manière complétement automatique chaque nuit. Il ne vous reste plus qu'à regarder les résultats dans votre navigateur préférée de temps en temps.

    5 - Remerciements

Si vos stats sont visibles sur le Net, une simple URL est la bienvenue mais si elles ne sont pas accessibles, un petit email me fera bien plaisir.
