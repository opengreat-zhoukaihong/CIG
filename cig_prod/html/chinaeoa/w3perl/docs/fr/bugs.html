Problèmes connus et limitations
Bugs
  • Sur les systèmes d'exploitation avec des noms de fichiers de longueur limité (Mac, Win3.11 ou Archimedes), vous ne pouvez pas utiliser le plus haut niveau de précision possible qui est le 4. Sous Archimedes, il est possible d'utilisez !X-Files ou mieux !RaFS qui permet de contrer cette limitation.

  • Votre fichier de log doit contenir au moins une requête par semaine. Si cela n'est pas le cas, il y aura des décalages de date.

  • La correspondance d'URL peut dans certains cas ne pas fonctionner correctement dans le cas où les fichiers HTML contiennent certaines horreurs (Bill, apprends le HTML stp !). Les répertoires des utilisateurs ne sont pas toujours scannés par le script (à corriger)

  • Le suivi des liens symboliques ne fonctionnent que dans certains cas (à corriger).

  • Les stats incrementales ne sont pas calculées lorsqu'on utilise un fichier de log compréssé réinitialisé à chaque début de mois (corrigé pour cron-inc).

    Si vous trouvez un bug, merci de m'envoyer les informations suivantes :

    1. L'URL de w3perl
    2. La version de W3Perl
    3. Quel est le script en cause et sa version (cron-xxx.pl -v)
    4. Votre fichier config.pl
    5. Le nom de votre serveur et son système d'exploitation
    6. La sortie du script

    • Limitations
  • W3Perl n'est pas complément optimisé pour l'utilisation de fichiers de log journaliers. Mais je connais au moins un serveur qui utilise w3perl dans ce contexte sans problème. Voir la documentation sur les fichiers de log pour savoir comment utiliser w3perl dans un tel cas.

  • W3Perl s'arrête si le format de la date courante n'est pas en anglais (solution : export LANG="eng" ou setenv LANG eng)

  • W3Perl produit des trous dans les pages sur les stats sur les pays.
    Mettez à jour le fichier pays-conv pour y ajouter de nouveaux pays.

  • W3Perl ne peut pas effectuer de stats sur plus de 120 fichiers de log compréssés ou 10 ans de log. Si vous utilisez un serveur web depuis plus de 10 ans, prévenez moi !

    N'hésitez pas à m'envoyer les problèmes que vous avez pu rencontrer.... y compris mes fautes d'ortographes !
