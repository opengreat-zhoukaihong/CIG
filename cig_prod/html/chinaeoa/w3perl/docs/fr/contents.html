Scripts W3Perl
Tous les scripts inclus dans l'application w3perl sont listés avec leur fonction première. La plupart des scripts sont nécessaires au bon focntionnement de w3perl (optionnels sont en (*))

Fichiers distribués :

Installation et admininstration
install.pl Installation
config.pl Fichier de configuration (généré par l'interface d'administration)
Chaque script charge un fichier de configuration où se trouve vos préférences
cron-w3perl.pl Lance automatiquement via la crontab tous les scripts (installation) (*)
Les scripts seront lancés selon vos choix.
squeezelog Compresse vos fichiers de log chaque mois (*)
confstats.pl Génére vos fichiers de configuration (admin) (*)
Mis à jour et fabrication des fichiers de configuration par un formulaire

Scripts de stats
cron-pages.pl Initialisation des stats principales
cron-inc.pl Stats incrémentales
cron-hour.pl Stats sur les heures
cron-day.pl Stats sur les jours
cron-week.pl Stats sur les semaines
cron-month.pl Stats sur les mois
cron-url.pl Traduction d'URL et stats sur votre site Web (*)
cron-session.pl Stats sur les sessions(*)
cron-refer.pl Stats sur les références (*)
cron-agent.pl Stats sur les fureteurs (*)
cron-error.pl Stats sur les erreurs (*)

Documentations et fichiers de ressources
Fichiers HTML Documentation en HTML (francaise et anglaise)
config files Exemple de fichiers de configuration
pays-conv Traduction des codes de pays
resources Icons et fichiers de traduction
