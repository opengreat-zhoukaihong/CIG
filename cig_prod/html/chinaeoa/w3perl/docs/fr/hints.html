Installation :

  • Allez voir la documentation sur la mise à jour si vous désirez installer une version plus récente de l'application
  • Choississez avec attention les options de configuration dans le fichier de config.
  • Exécuter les scripts sur un petit fichier de log afin de vérifier que toutes les sorties correspondent à vos souhaits.
  • Si vous lancez par erreur plusieurs fois le même script, ne vous inqui´tez pas car cela n'a pas d'incidence....vous aurez simplement un message d'erreur.

Configuration :

  • Des fichiers de configurations sont disponibles pour Macintosh, RiscOS dans le répertoire /doc/conf/. Attention ceux-ci ne sont pas toujours très récents.
  • Si vous utilisez plusieurs serveurs sur une même machine, inutile d'installer l'application plusieurs fois, il vous faudra juste créer autant de fichiers de configuration config-.pl que de serveurs Web et d'appeler ces fichiers avec l'option -c pour les charger.
  • Si vous désirez exclure les stats sur vos utilisateurs, utilisez @exclurepert = ('/~'); dans le fichier de configuration. Toutes les url commençant par /~ seront excluses.
  • Profitez de la possibilité d'utiliser les expressions régulières dans le fichier de configuration. Par exemple, si vous désirez exclure les utilisateurs d'un fournisseur d'accès internet, utilisez @nolog = ('/ppp*.club-internet.fr/');
    (les gens se connectant depuis club-internet ont une adresse IP dynamique ppp<string>.club-internet.fr
  • Ne sélectionnez pas @selecrepert = ('/'); ! Cela doit correspondre à des sous-répertoires situés au même niveau.
  • Si vous avez un gros fichier de log et que seules les dernières journées vous interessent, utilisez les options disponibles dans cron-pages.pl. Il est possible de choisir la date de début et de fin.
  • N'oubliez pas d'utiliser cron-xxx.pl -h pour une liste des options possibles.
  • Si votre fichier de log n'est pas mis à jour de manière régulière (et pas en temps réel comme la plupart le font), le meilleure moment pour lancer les stats sur cron-inc est juste avant minuit.

Definitions
Site Le nombre de site différent s'étant connecté pendant un certain intervalle de temps.
Nouveau site Le nombre de site différent n'ayant jamais été vu par les scripts dans une certaine période de temps.
Domaine Identique à Site sauf qu'il s'agit de site interne au domaine local.
Requêtes Toutes les requêtes au serveur, incluant les cgi, images, pages html...
Accès Comptage uniquement des pages HTML
Requêtes du domaine Identique à Requêtes sauf qu'il s'agit de requêtes internes au domaine local.
