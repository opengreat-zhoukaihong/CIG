Compressed logfile

Most webmaster use compressed logfiles to save disk space (compression rate is around 90% for logfiles). Select 'Yes' in order to use this feature. The package will search for compressed logfiles first when reading files.

Binary tool

Give the path to your compression tool on your disk. NT users should give the path to their 'unzip' binary while Unix users will prefer their 'gzip' tool.

Compression suffix

If you are using zip file, standard extension for log filename is .zip, if using gzip, log filename extension is .gz.
If you need support for other compression tools, ask me and I will add them in the package.
