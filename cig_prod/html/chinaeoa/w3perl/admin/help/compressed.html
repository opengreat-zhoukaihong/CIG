Log filename
Splited log filename

Most webmaster split their logfiles each day or each month. The package is able to read such files. You'll need to specify the structure of your log filename.

Field Description Value (ex)
%prefixlog This is the filename core, i.e the constant string of your log filename. access
%day Day in the 01-31 range 12
%month Month in the 01-12 range 10
%lettermonth Month in the Jan-Dec range (first letter uppercase) Nov
%year Year in an 4 digits number 2000
%smallyear Year in an 2 digits number 00
