Configuration - Step 1
Choose the name of your configuration file where your data will be saved. If default is selected, the filename will be config, if not, you'll have to use the -c <config filename> flag to load your configuration file. Don't write file extension with your filename, it will be added by the script.

Choose the operating system where w3perl will be used and the name of the httpd server you are using.
