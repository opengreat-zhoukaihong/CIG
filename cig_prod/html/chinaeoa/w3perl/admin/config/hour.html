  Flag Description Value
-a stats over full logfile  
-i <file> input logfile

-p precision level  minimal  basic  medium  high
-l language output  french  english  dutch  japanese  spanish  finnish
-r <file> string filter on files
-j <file> stats on dd/Mmm/yyyy / /
-s Hosts limit for display
-t Display toplist files
