Hourly or daily crontab use

If you want to have stats updated each hour, you'll need to launch your crontab every hour. If you select to run only once a day your hourly stats (midnight is best), then your crontab need to be run only once a day.

Multiple web server

If you need to run stats over multiple server, don't run everything at the same time ! For example, run the first scheduler for your first webserver at 00 h 10, the second ten minutes later and so on.
