Incremental stats
This script will update your main stats area. You can choose to alter few values as :
  Flag Description Value
-s <nbdays> scan for the nbdays previous days
-i <file> input logfile
-p precision level  minimal  basic  medium  high
-l language output  french  english  dutch  japanese  spanish  finnish  german
-j <date> stats for dd/Mmm/yyyy / /
-t Display toplist files
